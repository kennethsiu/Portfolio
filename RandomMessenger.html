RandomMessenger

RandomMessenger is a mobile application created by seven developers, including me, with an intent to connect students across campus. UC San Diego is notoriously known for being "socially dead," so we wanted to create a social media platform to improve this stereotype.

This application was developed on Android Studio, using Java for the back-end programming language and Google Firebase as the database to store the user messages. To ensure that this application connected only UCSD students, we created an e-mail verification system which sent an e-mail to your @ucsd.edu e-mail.



Upon verifying your account through e-mail, users were brought to a page where they could enter in their interests. Interests were optional, but recommended to connect students with similar hobbies.

Once interests were inputted, users could click "Start Chat" which would connect you with another random user on campus who was also using the application with similar interests. If users enjoyed their conversation and were interested in continuing the conversation, they could share contact information through message. One additional feature that we would like to implement is to add a button on the messaging page that shares User A's info with User B once the button is pressed.

Otherwise, if the user did not enjoy the chat, there was also an option to leave the conversation by clicking "Exit Chat." This would take you back to the homepage to start a new conversation.
